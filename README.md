# 🧮 Scientific Calculator using Python Tkinter

A GUI-based Scientific Calculator developed using Python and Tkinter. This application provides both basic arithmetic operations and scientific functions through a clean dark-themed interface.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🔢 Decimal Number Support
- √ Square Root
- x² Square Calculation
- % Percentage Calculation
- ⌫ Delete / Backspace Function
- 🧹 Clear Function
- 🌙 Dark Theme User Interface
- 🎨 Custom Calculator Icon

## 🛠 Technologies Used

- Python
- Tkinter
- Math Module

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Scientific-Calculator-Tkinter.git
```

2. Navigate to the project folder

```bash
cd Scientific-Calculator-Tkinter
```

3. Run the application

```bash
python calculator.py
```
